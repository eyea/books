# books
一个简单程序员的日常读书笔记

最近神经发疯的花了大几百大洋买了近20本书，电子书架也添加了不少新书，再看看自己的云盘，各类书籍不少，看的不多。

结合当前肚子里的墨水只有可怜的59ml，生活、工作的迫切需要储备以及发挥，故开始啃书。

此项目，作为个人阅读记录使用，如果有缘的你也看到了其中某一本书，可到 [issues](https://github.com/eyea/books/issues) 相关话题下 聊聊 :)
